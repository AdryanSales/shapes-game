# Jogo das Formas

Jogo educativo multijogador de encaixe de formas geométricas, feito para crianças.

**Jogar agora:** [adryansales.github.io/shapes-game](https://adryansales.github.io/shapes-game/)



## Como jogar

O jogo é para **2 jogadores** que se alternam na mesma tela.

1. **Role o dado** — clique no dado para sortear uma forma geométrica (círculo, triângulo, quadrado, retângulo, paralelogramo ou semicírculo).
2. **Arraste a forma** — a forma sorteada aparece acima ou ao lado do dado; arraste-a até o encaixe correto em um dos dois desenhos.
3. **Marcar pontos** — o jogador que encaixar a última peça que completa o desenho, marca um ponto.
4. **Encaixe errado** — a peça volta ao lugar e a vez passa para o outro jogador.
5. **Complete os desenhos** — há dois desenhos para montar: uma **Casa** e um **Rosto**. O jogo termina quando ambos estiverem completos.
6. **Vence quem completar mais desenhos** ao final.

### Formas e encaixes

| Forma | Onde encaixa |
|---|---|
| Paralelogramo | Telhado inclinado da casa |
| Triângulo | Ponta do telhado / nariz do rosto |
| Retângulo | Parede da casa |
| Quadrado | Porta da casa |
| Círculo | Olhos do rosto |
| Semicírculo | Sobrancelhas e boca do rosto |
